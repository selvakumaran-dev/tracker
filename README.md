## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment to Vercel

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Import your repository
4. Vercel will automatically detect Next.js and deploy

Or use the Vercel CLI:
```bash
npm i -g vercel
vercel
```

## Usage

1. **Add Expenses**: Click the floating "+" button to add a new expense
2. **View Statistics**: Check the stats cards at the top for quick insights
3. **View Charts**: Scroll down to see visual representations of your spending
4. **Enable Notifications**: Click "Enable Notifications" to get daily reminders
5. **Delete Expenses**: Click the trash icon next to any expense to remove it
